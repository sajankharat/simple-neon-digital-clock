
# Neon Digital Clock

A simple digital clock with a neon aesthetic that displays the current time and date in real-time.

## Features

- **Real-time Clock Display**: Shows hours, minutes, and seconds in HH:MM:SS format
- **Date Display**: Displays the current date below the time
- **Neon Styling**: Eye-catching neon visual effects with glowing text
- **Responsive Design**: Works on different screen sizes

## Project Structure

- `index.html` - Main HTML file with clock display elements
- `styles.css` - Styling for neon effects and layout
- `app.js` - JavaScript logic to update time and date in real-time

## How to Use

1. Open `index.html` in a web browser
2. The clock will automatically display the current time and date
3. The time updates every second

## Technical Details

- Built with vanilla HTML, CSS, and JavaScript
- No external dependencies required
- Uses DOM manipulation to update time and date dynamically
